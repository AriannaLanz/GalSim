const FluxData* galsim::ProbabilityTree< FluxData >::Element::find ( double &  cumulativeFlux) const

Recursive routine to find Element that contains a given value in the cumulative flux (unnormalized probability) distribution.

Parameters:
[in,out]cumulativeFluxOn input, randomly chosen point on cumulative flux distribution. On output: fraction of chosen member's flux that is below the input value on cumulative flux distribution.
Returns:
pointer to member that contains input cumulative flux point.
