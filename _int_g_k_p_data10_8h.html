Detailed Description

Gauss-Kronrod-Patterson quadrature coefficients using the 10-, 21-, 43-, 87- and 175-point rule.

The coefficients up through the 87-point rule were calculated with 101 decimal digit arithmetic by L. W. Fullerton, Bell Labs, Nov 1981. The 175-point rule was calculated by Mike Jarvis using long double precision.
