30 /*
31  * There are three levels of verbosity which can be helpful when debugging, which are written as
32  * dbg, xdbg, xxdbg (all defined in Std.h).

34  *
35  * If DEBUGLOGGING is #defined, then these write out to *dbgout, according to the value of
36  * verbose_level.
37  * dbg requires verbose_level >= 1
38  * xdbg requires verbose_level >= 2
39  * xxdbg requires verbose_level >= 3
40  * If DEBUGLOGGING is not defined, the all three becomes just `if (false) std::cerr`,
