double galsim::SBAiry::SBAiryImpl::AiryInfoNoObs::kValue ( double  ksq_over_pisq) const
virtual

Returns the k-space value of the Airy function.

Parameters:
[in]ksq_over_pisqshould be given in units of lam_over_D (i.e. k_true^2 / (pi^2 * D^2))

This is used to calculate the real kValue, but it comes back unnormalized. The value at k=0 is Pi * (1-obs^2), so the value needs to be multiplied by flux / (Pi * (1-obs^2)).
