Detailed Description

A class storing the data from an input catalog.

Each row corresponds to a different object to be built, and each column stores some item of information about that object (e.g. flux or half_light_radius).

Parameters:
file_nameFilename of the input catalog. (Required)
dirDirectory catalog is in.
file_typeEither 'ASCII' (currently the only, default, option) or (soon) 'FITS'. (TODO: default = determine from extension or, if that fails, ASCII)
commentsThe character used to indicate the start of a comment in an ASCII catalog. (default='#').
