Build a GSObject using config dict for key=key.

Parameters:
configA dict with the configuration information.
keyThe key name in config indicating which object to build.
baseA dict which stores potentially useful things like base['rng'] = random number generator base['catalog'] = input catalog for InputCat items base['real_catalog'] = real galaxy catalog for RealGalaxy objects Typically on the initial call to BuildGSObject, this will be the same as config, hence the name base.
gsparamsOptionally, provide non-default gsparams items. Any gsparams specified at this level will be added to the list. This should be a dict with whatever kwargs should be used in constructing the GSParams object.
Returns:
gsobject, safe gsobject is the built object safe is a bool that says whether it is safe to use this object again next time
