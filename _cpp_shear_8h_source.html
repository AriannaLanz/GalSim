00039 // Shear is represented internally by e1 and e2, which are the second-moment
00040 // definitions: ellipse with axes a & b has e=(a^2-b^2)/(a^2+b^2).
00041 // But can get/set the ellipticity by two other measures:
00042 // g is "reduced shear" such that g=(a-b)/(a+b)
00043 // eta is "conformal shear" such that a/b = exp(eta).
00044 // Beta is always the position angle of major axis.
