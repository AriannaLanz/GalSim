40 // Shear is represented internally by e1 and e2, which are the second-moment
41 // definitions: ellipse with axes a & b has e=(a^2-b^2)/(a^2+b^2).
42 // But can get/set the ellipticity by two other measures:
43 // g is "reduced shear" such that g=(a-b)/(a+b)
44 // eta is "conformal shear" such that a/b = exp(eta).
45 // Beta is always the position angle of major axis.
