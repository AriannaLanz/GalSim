Build a number of postage stamp images as specified by the config dict.

Parameters:
nstampsHow many stamps to build
configA configuration dict.
xsizeThe size of a single stamp in the x direction
ysizeThe size of a single stamp in the y direction
nprocHow many processes to use.
do_noiseWhether to add noise to the image (according to config['noise'])
loggerIf given, a logger object to log progress.
make_psf_imageWhether to make psf_image
make_weight_imageWhether to make weight_image
make_badpix_imageWhether to make badpix_image
Returns:
(images, psf_images, weight_images, badpix_images) (All in tuple are lists) 
input is a queue with (args, info) tuples:
    kwargs are the arguments to pass to BuildSingleStamp
    info is passed along to the output queue.
output is a queue storing (result, info, proc) tuples:
    result is the returned tuple from BuildSingleStamp: 
        (image, psf_image, weight_image, badpix_image, time).
    info is passed through from the input queue.
    proc is the process name.

Process the output field, building and writing all the specified image files.

input is a queue with (args, info) tuples: kwargs are the arguments to pass to build_func info is passed along to the output queue. output is a queue storing (result, info, proc) tuples: result is the returned value from build_func (file_name, time). info is passed through from the input queue. proc is the process name.
