double galsim::KolmogorovInfo::kValue ( double  ksq_over_pisq) const

Returns the k-space value of the Kolmogorov function.

Parameters:
[in]ksq_over_pisqshould be given in units of lam_over_r0 (i.e. k_true^2 / (pi^2 * r0^2))

This is used to calculate the real kValue, but it comes back unnormalized. The value at k=0 is Pi, so the value needs to be multiplied by flux / Pi.
