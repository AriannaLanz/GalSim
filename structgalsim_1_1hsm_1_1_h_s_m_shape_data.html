Detailed Description

Struct containing information about the shape of an object.

This hsm representation of an object shape contains two Shear objects, one for the observed shape and one after PSF correction. It also contains information about what PSF correction was used; if no PSF correction was carried out and only the observed moments were measured, the PSF correction method will be 'None'.
