Parameters
nobjectsHow many postage stamps to build.
configA configuration dict.
nprocHow many processes to use. [default: 1]
loggerIf given, a logger object to log progress. [default: None]
obj_numIf given, the current obj_num. [default: 0]
xsizeThe size of a single stamp in the x direction. [default: 0, which means to look for config.image.stamp_xsize, and if that's not there, use automatic sizing.]
ysizeThe size of a single stamp in the y direction. [default: 0, which means to look for config.image.stamp_xsize, and if that's not there, use automatic sizing.]
do_noiseWhether to add noise to the image (according to config['noise']). [default: True]
make_psf_imageWhether to make psf_image. [default: False]
make_weight_imageWhether to make weight_image. [default: False]
make_badpix_imageWhether to make badpix_image. [default: False]
Returns
